## Quick start

```ts
import { createI18n } from "@echojs-ecosystem/i18n";

const i18n = createI18n({
  locale: "en",
  messages: { en: () => import("./locales/en.json") },
});
```

> [!tip]
> Wire via `createI18nProvider` in `createEchoApp` — see `apps/docs/src/core/providers/i18n.ts`.
