## Key APIs

| Export | Role |
| ------ | ---- |
| [`createI18n`](/docs/packages/i18n/api/create-i18n) | Message catalog, `t()`, locale signals |
| [`createI18nProvider`](/docs/packages/i18n/api/create-i18n-provider) | App-level provider for `createEchoApp` |
| [`detectLocale`](/docs/packages/i18n/api/detect-locale) | Browser / cookie locale detection |

## Common patterns

- Lazy-load locale JSON with **dynamic `import()`** in the `messages` map.
- Use **`() => i18n.t('key')`** in HyperDOM views so locale switches re-render.
- Wire via **`createI18nProvider`** — see `apps/docs/src/core/providers/i18n.ts`.

> [!tip] Wire via `createI18nProvider` in `createEchoApp` — see `apps/docs/src/core/providers/i18n.ts`.
